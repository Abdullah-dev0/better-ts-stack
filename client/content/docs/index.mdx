## Welcome to Better-TS-Stack

Better-TS-Stack is a powerful CLI tool that generates fully configured TypeScript projects with backend, frontend, database integration, Docker support, and more through an interactive setup.

<Callout type="info" title="Quick Start">
  Get your project running in under 5 minutes with our interactive CLI.
</Callout>

## Quick Installation

<Steps>

<Step>
### Install the CLI

<Tabs items={['npm', 'pnpm', 'yarn']}>
  <Tab value="npm">
    ```bash
    npm install -g better-ts-stack
    ```
  </Tab>
  <Tab value="pnpm">
    ```bash
    pnpm add -g better-ts-stack
    ```
  </Tab>
  <Tab value="yarn">
    ```bash
    yarn global add better-ts-stack
    ```
  </Tab>
</Tabs>

</Step>

<Step>
### Create Your Project

Run the interactive CLI to generate your project:

```bash
better-ts-stack create my-awesome-app
```

The CLI will guide you through selecting:
- Project name and description
- Backend framework (Express, Fastify, or none)
- Frontend framework (Next.js, React, or none)
- Database (PostgreSQL, MySQL, or none)
- Additional features (Docker, testing, etc.)

</Step>

<Step>
### Start Development

Navigate to your project and start coding:

```bash
cd my-awesome-app
npm install
npm run dev
```

</Step>

</Steps>

## What's Included?

Your generated project includes:

- **Backend API** - RESTful API with TypeScript, validation, and error handling
- **Frontend App** - Modern React/Next.js application with routing
- **Database Setup** - Configured ORM (Prisma/TypeORM) with migrations
- **Docker Compose** - Multi-container orchestration for development and production
- **CI/CD Pipeline** - GitHub Actions workflow template
- **Documentation** - README with project-specific instructions

<Callout type="warn" title="Requirements">
  Node.js 18+ is required. Make sure you have it installed before proceeding.
</Callout>
