## What is Better-TS-Stack?

Better-TS-Stack is an interactive CLI tool that generates fully configured TypeScript projects with your choice of backend, frontend, database, and DevOps tools. No configuration files to wrestle with‚Äîjust answer a few prompts and get a production-ready codebase.

<Callout type="info" title="Perfect for">
	- Starting new full-stack TypeScript projects - Prototyping APIs with type-safe backends - Learning modern TypeScript
	patterns - Standardizing project structure across teams
</Callout>

## Quick Start

Get a new project running in under 2 minutes:

<Steps>

<Step>
### Run the CLI

<Tabs items={['npx (recommended)', 'Global Install', 'pnpm', 'bun']}>
  <Tab value="npx (recommended)">
    ```bash
    npx better-ts-stack
    ```
  </Tab>
  <Tab value="Global Install">
    ```bash
    npm install -g better-ts-stack
    better-ts-stack
    ```
  </Tab>
  <Tab value="pnpm">
    ```bash
    pnpm dlx better-ts-stack
    ```
  </Tab>
  <Tab value="bun">
    ```bash
    bunx better-ts-stack
    ```
  </Tab>
</Tabs>
</Step>

<Step>
### Answer the Prompts

The CLI will guide you through:

1. **Project name** ‚Äî What to call your new project
2. **Application type** ‚Äî Backend API only, or full-stack with frontend
3. **Backend framework** ‚Äî Express (NestJS coming soon)
4. **Database** ‚Äî PostgreSQL with Prisma, MongoDB with Mongoose, or Drizzle
5. **Authentication** ‚Äî Optional JWT auth for Express or Better Auth for Next.js
6. **Docker** ‚Äî Optional Docker Compose setup
7. **Package manager** ‚Äî npm, pnpm, or bun

</Step>

<Step>
### Start Developing

```bash
cd your-project-name
npm install  # or pnpm install, bun install
npm run dev
```

Your project is now running with hot reload enabled!

</Step>

</Steps>

## Requirements

- **Node.js** 18.0.0 or higher
- **Package manager** ‚Äî npm, pnpm, or bun
- **Git** ‚Äî For repository initialization (optional)

<Callout type="warn">Make sure you have Node.js 18+ installed. Check with: `node --version`</Callout>
